# ☕ Sistem Manajemen Kafe

> Website manajemen untuk coffee shop dengan 20+ halaman. Dibangun dengan HTML, CSS, dan JavaScript murni. Proyek portofolio skala menengah yang saya kerjakan untuk menunjukkan kemampuan front-end saya.

---

## 📋 Fitur

| Fitur | Deskripsi |
|-------|-----------|
| **Admin Dashboard** | Pantau penjualan harian, pesanan aktif, dan stok produk |
| **Manajemen Produk** | Tambah, edit, hapus, dan cari menu kopi & makanan |
| **Tracking Order** | Lacak status pesanan dari "diproses" hingga "selesai" |
| **Login System** | Autentikasi untuk Admin dan Super Admin |
| **Katalog Kopi** | Tampilan menu interaktif untuk pelanggan |
| **Super Admin** | Kelola akun admin dan lihat laporan menyeluruh |

**Total Halaman:** 20+ halaman

## 🚀 Cara Menjalankan

### Prasyarat
- Browser modern (Chrome, Firefox, Edge)
- (Opsional) VS Code dengan Live Server

### Langkah
1. Clone repositori:
   ```bash
   git clone https://github.com/Zeeniac/Sistem-Manajemen-Kopi.git
   cd Sistem-Manajemen-Kopi
   ```
2. Buka file `index.html` di browser
3. Atau gunakan Live Server di VS Code (klik kanan → Open with Live Server)
